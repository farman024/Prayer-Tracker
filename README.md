### **✨ Features**
- **Daily 5 Prayer Checklist** - Fajr, Dhuhr, Asr, Maghrib, Isha with auto-save
- **Auto Hijri + Gregorian Date** - Shows `18 Shawwāl 1447 AH` alongside Gregorian
- **Streak Counter** - Tracks consecutive days. Don't break the chain
- **Monthly Calendar View** - Visual heatmap of your consistency
- **Export Progress** - Download JSON backup of all your data
- **100% Offline PWA** - Service worker caches everything. Works without internet
- **Global Prayer Times** - Auto-detects location via Aladhan API. Falls back to Mecca
- **No Login, No Ads, No Tracking** - Your data never leaves your browser
- **Hijri Guide** - **[📄 Download Hijri Calendar 101 PDF](./Hijri-Guide.pdf)** to understand Islamic dates

### **📦 Deploy Your Own in 2 Minutes**
1. **Fork this repo** → Click Fork button top right
2. **Enable Pages** → Settings → Pages → Deploy from `main` branch → Save
3. **Done** - Your tracker is live at `https://yourusername.github.io/prayer-tracker`

To customize: Edit `index.html`. Change colors, add Tasbeeh counter, set default city.
